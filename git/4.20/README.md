### ファイルの変更内容を取り消す
・git checkout <ファイル名/ディレクトリ名>

### ステージした変更内容を取り消す
 ・　git reset HEAD <ファイル名/ディレクトリ名>
前変更を取り消す
git reset HEAD.

ステージした変更分だけを取り消すのでワークツリーにはんも影響を与えない。

### 直前のコミットをやり直す
```git commit --amend```

・注意点
　```リモートリポジトリにpushしたコミットはやり直してはダメ```
 
 
### リモートリポジトリ
リモートリポジトリ... オンライン上にあるリポジトリ
ローカルリポジトリからリモートリポジトリに送ることで他人もおsのリポジトリを使用できる。

 ```git remote```
